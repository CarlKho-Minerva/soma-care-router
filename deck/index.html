Architecture

Built for Production

  • Gemini 3 — Agentic reasoning via Google Cloud Agent Builder (ADK)
  • MongoDB Atlas — Provider database with vector search + Atlas Search
  • MCP Protocol — MongoDB MCP server for structured tool calls
  • Privacy Bridge — PII stripping happens on-device before any cloud call
  • Health Passport — Gemma 4 E2B + PaddleOCR, local-first vault
  • Web UI — Real-time agent trace visualization
